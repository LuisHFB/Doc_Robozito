# Montagem

Para a montagem precisaremos de:

|     **Quantidade** | **Componente**             | **Método de Fabricação**    |
| :----------------: | :------------------------  | --------------------------: |
|                  3 | Parafuso M3 x 16mm         | Comprar                     |
|                  3 | Porca M3                   | Comprar                     |
|                  8 | Parafuso M2 x 8mm          | Comprar                     |
|                  8 | Porca M2                   | Comprar                     |
|                  1 | Suporte Motor Esquerdo     | Impressão 3D                |
|                  1 | Suporte Motor Direito      | Impressão 3D                |
|                  2 | Rolamento MR 126 ZZ        | Comprar                     |
|                  2 | Rodas                      | Impressão 3D                |
|                  2 | Suporte Roda               | Impressão 3D                |
|                  1 | Apoio Frontal              | Impressão 3D                |
|                  4 | Micro Servo 9g SG90        | Comprar                     |
|                  1 | Corpo                      | Corte Laser                 |
|                  2 | Abraçadeira Nylon 95x2,4mm | Comprar                     |
|                  4 | Jumper fêmea-macho         | Comprar                     |
|                    |                            |                             |
